🔄 Plan de Regresión

⚙️ Ejecutar después de cada cambio mayor

Flujo mínimo para verificar que nada se rompió:

  1. Menú → Creación: Seleccionar Zancudo, nombre "Test", arquetipo Estratega → INOCULAR
  2. Mapa: Moverse en las 4 direcciones. Verificar límites. Acercarse a un spot.
  3. Spot → Combate: Cazar → pelear → ganar. Verificar loot + EXP + bestiario.
  4. Inventario: Abrir (I), equipar arma, usar consumible, verificar confirmación.
  5. Combate social: Iniciar combate → Danza → elegir postura → verificar resolución.
  6. Guardar/Cargar: F5 → cerrar → F9 → verificar estado.
  7. Fin demo: Craftear Anclaje → pantalla fin → volver al menú.
