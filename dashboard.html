📁 Estructura del Proyecto

descarada/
├── main.tscn              ← Escena principal (construida por código)
├── menu.tscn              ← Menú principal
├── creacion.tscn          ← Creación de personaje
├── assets/sprites/        ← Sprites del zancudo + spots
├── scripts/
│   ├── game_manager.gd    ← Autoload: stats, esencia, inventario, zona_origen
│   ├── loot_system.gd     ← Autoload: loot aleatorio por tier
│   ├── save_system.gd     ← Autoload: guardar/cargar (F5/F9)
│   ├── main.gd            ← Construye escena + maneja transiciones
│   ├── iso_map.gd         ← Laboratorio isométrico (12×12)
│   ├── zona_base.gd       ← Clase base para zonas (puertas, spawn, tiles)
│   ├── zona_pasillo.gd    ← El Pasillo (segunda zona)
│   ├── player.gd          ← Movimiento + detección de puertas
│   ├── combat_manager.gd  ← Combate por turnos
│   ├── hud.gd             ← Barras de stats
│   ├── spot.gd            ← Puntos de interacción
│   └── touch_controls.gd  ← Joystick virtual Android
└── export/                ← APKs exportados (v0.3.0)

la-descarada/ (rama experimental — mapa iso refactorizado)
├── scripts/
│   ├── iso_map.gd         ← Versión simplificada del laboratorio
│   ├── laboratorio.gd     ← Escena laboratorio con HUD
│   └── player.gd          ← Movimiento isométrico flotante
└── scenes/world/
    └── laboratorio.tscn   ← Escena con IsoMap + Player + Spots
